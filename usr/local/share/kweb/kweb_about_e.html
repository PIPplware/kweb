Keyboard Command Editor

Select Keyboard Shortcut:
Enter up to 4 URLs, keyboard or text commands or command lines:




By sending an empty form the command will be disabled.
The first URL will be used for redirection. All other URLs will be opened in a new window.
Keyboard commands:
?value
Value may be any combination of the following keyboard shortcuts: +-zfkmcqgteduwxy
Possible text commands:
)URL    (open URL in new window)
))URL    (open URL in new browser instance)
$user agent string (no argument = default)
!spellchecking language (e.g. "en_GB")
@encoding (set default encoding, no argument = utf-8)
Command Lines:
A command line should be entered like in a terminal. You may use it to start any external program.
The autoconfig command will be executed when the browser starts.
You'll find more information and some examples in the manual.
